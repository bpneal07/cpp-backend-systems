# C++ Backend & Systems Plaground

## Overview
This repository is a long-term learning project focused on backend and systems
programming using modern C++ on Linux.

The goal is to slowly build a production-style server while learning:
- Linux networking and system calls
- Concurrency and synchronization
- Event-driven IO (epoll)
- Performance measurement and optimization
- Reliability and failure handling

## Non-Goals
- No frontend
- No frameworks

## Current Status
- Project initialized
- No server implementation yet

## Build
Build instructions will be added once the first C++ code exists.

## Notes
This project is intentionally developed at a slow pace (~1–2 hours/week)
alongside college coursework.
